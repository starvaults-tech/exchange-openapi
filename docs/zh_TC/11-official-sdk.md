# 錯誤碼

返回報錯一般由兩個部分組成：錯誤碼和錯誤信息。錯誤碼是通用的，但是錯誤信息會有所不同。如下是一個報錯JSON Payload示例：

```java
{
  "code":-1121,
  "msg":"Invalid symbol."
}
```
